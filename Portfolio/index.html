Dinner and a Movie Web App using Restful APIs
Technologies used: Restful APIs, jQuery, JavaScript, HTML, CSS, Visual Studio Code & GitHub

Dinner and a Movie is a responsive web app that helps users decide where to order dinner and which movie to watch for a fun night in. Users input their location and it is sent to the EatStreet API Endpoint to find restaurants that provide food delivery or pickup nearby. Users will then navigate to a screen where they can input a movie that they like so that they can receive similar movie recommendations using the TasteDive Movie API Endpoint
